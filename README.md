# Car Retrieval & Classification System

Admission test project for Computer Vision position.

## Project Structure
- `data/`: Dataset storage (Images & Labels).
- `models/`: Trained model weights saved here.
- `src/pipeline.py`: Main inference script integrating Object Detection + Classification.
- `Vehicle_lama_tapi_berhasil.ipynb`: Main notebook for Training, Experiments, and Evaluation.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Place data in `data/`
3. Run notebook for training.
4. Run `src/pipeline.py` for inference.
